Markdown is a really powerfull technology, it's a great data-format for docs. In this article, I go through how to implement 
this  technology in your apps, easily and fast. This is a simple guide for [nextjs](https://nextjs.org).

Working with Markdown in NextJs is not very easy. Unlike other frameworks like nuxt, next provides not an option to use
Markdown directly.
